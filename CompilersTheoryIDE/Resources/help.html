Тема 51. Комментарии языка Python.

Грамматика

Определим грамматику комментариев языка Python G[‹Comment›] в нотации Хомского с продукциями P:
1.	‹Comment› → ‹SingleLineComment› | ‹MultiLineDoubleQuotesComment› | ‹MultiLineSingleQuotesComment›
2.	‹SingleLineComment› → «#»‹SymbolSequence›‹NewLine›
3.	‹MultiLineDoubleQuotesComment› → «"""»‹SymbolSequence›«"""»
4.	‹MultiLineSingleQuotesComment› → «'''»‹SymbolSequence›«'''»
5.	‹SymbolSequence› → ‹Symbol›{‹Symbol›}
6.	‹NewLine› → «;»
• ‹Symbol› → любой другой символ из набора символов языка Python

Следуя введенному формальному определению грамматики, представим G[‹Comment›] ее составляющими:
• Z = ‹Comment› 
• VT = {#, ", ', ;} ∪ набор символов языка Python 
• VN = {‹Comment›, ‹SingleLineComment›, ‹MultiLineDoubleQuotesComment›, ‹MultiLineSingleQuotesComment›, ‹SymbolSequence›, ‹NewLine›}

Классификация грамматики

Грамматика является автоматной, (по классификации Хомского)

Метод анализа

Метод Айронса

Разрабатываемый синтаксический анализатор построен на базе автоматной грамматики. При нахождении лексемы, которая не соответствует грамматике предлагается свести алгоритм нейтрализации к последовательному удалению следующего символа во входной цепочке до тех пор, пока следующий символ не окажется одним из допустимых в данный момент разбора.

Этот алгоритм был мной уже реализован в Лабораторной работе №3. В таблице ошибок выводятся их местоположение и текст ошибки, содержащий информацию об отброшенном фрагменте.
